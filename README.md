## 🚀 Key Features

### 🎮 Core Gameplay
- **Rogue-lite Progression**: Level up during runs to choose from a variety of powerful modules.
- **Meta-Progression**: Use salvaged **Data Scraps** to permanently upgrade your hull, weapons, and core systems in the hangar.
- **High-Stakes Combat**: Face off against Swarmers, Tanks, Shooters, and the terrifying Void Bosses.

### 🛠️ Advanced Mechanics
- **Phase Shift (Dash)**: Blink through space to avoid damage and reposition.
- **Sentry Drones**: Deploy orbiting drones for automated defense.
- **Singularity Shots**: Manipulate gravity to clump enemies together for massive area damage.
- **Dynamic Music**: A procedurally generated soundtrack that adapts to the intensity of the battle.

### 🎨 Visuals & Audio
- **Neon Aesthetic**: High-contrast visuals with bloom effects, dynamic nebulae, and parallax backgrounds.
- **Procedural Audio**: All sound effects and music are generated in real-time via the Web Audio API.
- **Smooth Performance**: Optimized for 60FPS on both Desktop and Mobile devices.

## 🕹️ How to Play

### Controls
| Action | Desktop | Mobile |
| :--- | :--- | :--- |
| **Move** | `WASD` / `Arrow Keys` | Virtual Joystick |
| **Aim** | Mouse Cursor | Automatic |
| **Dash** | `Space` / `Shift` | Automatic/Contextual |
| **Pause** | `Esc` | Menu Button |

### Goal
Survive as many waves as possible. Collect XP to level up and salvage data scraps upon death to buy permanent upgrades.
